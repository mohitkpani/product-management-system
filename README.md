# Product Management System

A full stack Product Management System developed using Spring Boot, Thymeleaf, MySQL, HTML, CSS, and Spring Security.

---

## Features

- User Login Authentication
- Add Product
- View All Products
- Search Product By ID
- Update Product
- Delete Product
- Form Validation
- Professional User Interface
- Logout Functionality

---

## Technologies Used

- Java
- Spring Boot
- Spring MVC
- Spring Security
- Thymeleaf
- MySQL
- HTML
- CSS

---

## Tools Used

- Spring Tool Suite (STS)
- MySQL Workbench
- GitHub
